# POC API BscScan

## Objective

This POC has as objective test the BscScan API.

## Stack

- express.js
- node.js
- TypeScript

## Details project

- Using **Express** to running the server
- Using **TypeScript** in `DEV` environment
- Using `ts-node-dev` with lib to running APP
- Using `dotenv` to hide sensitive information
